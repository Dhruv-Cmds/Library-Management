# 📚 Library Management System (Java)

A **console-based Library Management System** developed using **Core Java**.  
This project focuses on implementing **Object-Oriented Programming (OOP)** principles and basic library operations through a command-line interface.

---

## 🚀 Features

- Add new books
- View available books
- Manage library users
- Issue and return books
- Simple and user-friendly CLI
- Clean OOP-based structure

---

## 🛠️ Technologies Used

- Java (JDK 8+)
- Core Java
- Object-Oriented Programming (OOP)
- Command Line Interface (CLI)
